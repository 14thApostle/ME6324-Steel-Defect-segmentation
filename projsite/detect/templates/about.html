About the project

This website was created to work as a frontend for the project "Segmentation of faults in steel" that was created by our team for the course "Artificial Intelligence in Manufacturing" at the IIT Madras. The project was created by the following team members: - Irfaan Arif - ME18B048 - Shaloo Vardhini V - ME18B073 - Sabari Krishna M - ME18B071

The aim of our project was to localise and classify surface defects on a steel sheet using various deep learning algorithms. We employed semantic segmentation to successfully partition each digital image into multiple labelled segments which include - Pitted Surface - Scratches - Inclusion - Rolled in scale Our main evaluation metric is the Dice Coefficient which tracks the intersection over average area. The main dataset used for this project is the Severstal Steel defect dataset which is a publically available dataset on Kaggle. It can be found available at the following link: https://www.kaggle.com/c/severstal-steel-defect-detection/data

The model we have trained is a standard U-Net, it uses resnet18 as the encoder network and a decoder network. The weights were pre-trained on the ImageNet dataset. We used a 80:20 train:test split to train our model. We used a standard binary cross-entropy loss function and the Adam optimizer. We achieved a test score of 93.96% on the test set. Defect detection is one of the critical parts of manufacturing of a product in an industry. Manually inspecting the bulk production of steel sheets for defect detection is a tiresome task and could be prone to human errors. Analysis of the results show that the model we developed is promising for practical industry applications.
